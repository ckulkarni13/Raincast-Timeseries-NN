# RainCast-Multimodal: Dual-Branch Hybrid Neural Network

An advanced deep learning framework that fuses Computer Vision and Time-Series Analysis to predict urban rainfall. Processes lake surface imagery alongside historical meteorological data for 24, 48, and 72-hour forecasts.
